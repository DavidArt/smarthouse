                 SmartHouse
A web app for controlling your smart house.

Backend Technology Stack

Language: 		        Java 8 JDK
Framework: 		        Spring Boot because it has Tomcat embedded and simplifies the configuration
Build Tool:		        Maven
Rest web services:    	Jax-RS with Jersey implementation
Persistence: 		    MySQL and LiquiBase, Jpa and Hibernate
Patterns: 		        DAO, DTO
Log: 			        Lombok
Version Control System: Git
Mapper:                 Dozer
Others: 		        MySQLWorkbench and Postman
